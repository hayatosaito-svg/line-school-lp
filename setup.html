Claude Code セットアップガイド

所要時間：約5分

1

Claude「Pro」に契約する

以下のリンクから、真ん中のプラン（Pro）を選択して契約してください。

Claude 料金ページを開く
↓
2

「ターミナル.app」を開く

💻
ターミナル
Terminal.app

見つからない場合は ⌘ + Space を押して
「ターミナル」と入力すると出てきます。

開くと黒い画面（またはグレーの画面）が出ます。
そこにコマンドを入力していきます。

↓
3

コマンドを入力してインストール

1つ目：以下をコピーして、ターミナルに貼り付けて Enter

curl -fsSL https://claude.ai/install.sh | bash
↓

2つ目：以下をコピーして、ターミナルに貼り付けて Enter

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
↓

auth~~ みたいなのが表示されます（認証です）。
矢印キーで選んで Enter。

自動でWebが開くので、自分のClaudeアカウントを選択して進んでください。

↓
✓

設定完了！

🎉

セットアップ完了

以降、ターミナルで「claude」と入力するだけで
Claude Codeが起動します

claude

「Password:」と表示されたら

PCのパスワードを入力してください。
何も表示されませんが、そのまま入力して Enter で大丈夫です。

「Press RETURN/ENTER to continue...」と表示されたら

そのまま Enter を押してください。
